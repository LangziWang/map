Method Documentation

- (instancetype) initWithMapView: (MAMapView *)  mapView
delegate: (id< AMapNaviViewControllerDelegate >)  delegate 

AMapNaviViewController初始化方法，在传入mapView对象前，需要保存地图当前所有状态，导航SDK会重置mapView的所有状态。

注解
由于导航SDK会复用用户的MapView，所以在以下两种操作前需要保存地图当前所有状态，导航SDK会重置mapView的所有状态，操作包括：
1.在初始化AMapNaviViewController前
2.在通过AMapNaviManager去展示AMapNaviViewController前
将被重置的地图状态包括:
1.重置MapView的代理delegate；
2.清除所有的标注Annotation和覆盖物Overlay；
3.重置MapView的旋转角度RotationDegree和倾斜角度CameraDegree；
4.重置MapView的比例尺，指南针，地图类型MapType和交通图层的显示状态；
5.重置MapView的可视区域visibleMapRect；
6.将MapView从其原来的视图层级中移除，并加入到AMapNaviViewController的view层级中；
参数
mapViewMAMapView对象。
delegateAMapNaviViewController的代理。
返回
AMapNaviViewController类对象id（当AMapNavi.bundle没有正确导入时，返回nil）。
