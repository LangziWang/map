- (BOOL) calculateDriveRouteWithEndPoints: (NSArray *)  endPoints
wayPoints: (NSArray *)  wayPoints
drivingStrategy: (AMapNaviDrivingStrategy)  strategy 

以下算路方法需要高德地图坐标

不带起点的驾车路径计算

参数
endPoints终点坐标。支持多个终点，终点列表的尾点为实际导航终点，其他坐标点为辅助信息，带有方向性，可有效避免算路到马路的另一侧。
wayPoints途经点坐标序列，同时支持最多3个途经点的路径规划。
strategy驾车路径的计算策略。
返回
路径是否计算成功。

readwritenonatomicassign

探测模式，默认为AMapDetecteModeNone（需要联网）。

探测模式适用于不设置目的地的播报内容（包括：电子眼、特殊道路设施等等）提示。
